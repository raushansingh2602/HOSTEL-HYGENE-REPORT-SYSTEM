# Hostel Hygiene Report System – Project Statement

## 1. Problem Statement

          -Maintaining hygiene in hostels is challenging due to lack of proper monitoring.  
          -Students often have no quick way to report issues, delaying corrective action.  
          -This project provides a *digital solution* for students to report hygiene issues efficiently.

## 2. Scope of the Project

         - Collect hygiene feedback from students in a structured way  
         - Cover areas: Water Quality, Food Quality, Cleanliness, Kitchen Hygiene, Waste Management  
         - Allow general complaints and feedback submission  
         - Generate a complete report for hostel authorities  

      *Limitations:* 
         - Data is not saved permanently (no database)  
         - Future improvements could include web/mobile interface and persistent storage  

## 3. Target Users

        - *Primary:* Hostel students submitting reports  
        - *Secondary:* Hostel management/staff reviewing reports  

## 4. High-Level Features

        1. Student information input  
        2. Dynamic hostel block selection  
        3. Room number input  
        4. Hygiene category reporting  
        5. Rating system: Good / Average / Poor  
        6. Feedback/Complaint submission  
        7. Final hygiene report generation
